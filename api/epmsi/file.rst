/epmsi/:annee/:id/file/:type
============================


Description
-----------

Renvoi le fichier ZIP contenant les exports HTML de e-PMSI.

Type
----

GET

Paramètres
----------

========= ===========================================
Paramètre 
========= ===========================================
annee     Années supportées 2011, 2012.

type      exported_zip
