/epmsi/mat2a/had/:annee/M0/:id/status¶

Description¶

Statut du passage des fichiers sur e-PMSI (MAT2A HAD)

Type¶

GET

Paramètres¶

Paramètre  
annee Années supportées : 2011, 2012.
id id renvoyé par l’api /epmsi/mat2a/had/:annee/M0/send

Retour¶

Succès¶

Paramètre Libellé
status
  • SUCCESS
  • WAITING
  • RUNNING
  • ERROR

Exemple¶

Appel :

curl  http://autoih.localhost/api.php/epmsi/mat2a/had/2012/M0/69a7d9f2c4560159d39731b3c91515a5/status

Retour :

{"status":0,"message":"OK","content":{"status":"RUNNING"}}
