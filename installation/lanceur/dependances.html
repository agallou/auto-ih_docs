Installation dépendances¶

Il faut installer :

  • .NET Framework
  • PHP
  • Firefox
  • Sahi
  • GENRSA
  • PAPRICA

.NET Framework¶

Nécéssaire au fonctionnement de GENRSA.

.NET Framework 3.5 peut être téléchargé à cette adresse : http://www.microsoft.com/downloads/fr-fr/details.aspx?familyid=333325fd-ae52-4e35-b531-508d977d32a6.

PHP¶

Il est préférable de configurer la variable d’environnement PHP_PATH pour éviter d’avoir à la préciser lors du lancement du lanceur d’applications.

PHP peut être téléchargé à cette adresse: http://windows.php.net/download/.

Firefox¶

Permet de se connecter sur e-PMSI pour envoyer les fichiers exportés depuis GENRSA.

Sahi¶

Permet de simuler le comportement d’un utilisateur utilisant un navigateur (ici firefox) pour effectuer les envois sur la plateforme e-PMSI.

Sahi peut être téléchargé à cette adresse: http://sahi.co.in/w/.

GENRSA¶

GENRSA peut être téléchargé à cette adresse : http://download.atih.sante.fr/index.php?lettre=G.

PAPRICA¶

PAPRICA peut être téléchargé à cette adresse : http://download.atih.sante.fr/index.php?lettre=P.
